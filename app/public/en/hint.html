Show Menu Click menu key or scroll info bar at bottom, will show different menu when you select different thing or not select.
Card Desc Select a card by single click, then click info bar at bottom, return key to close the window.
Draw Drag deck to hand card.
Face Up Use Drag an face up card to field.
Face Down Use Drag a face down card to field.
Open/Close Double click the card on field or in hand.
Atk/Def Long press the card in monster zone.
To Graveyard Drag a card to graveyard.
Open Deck Double click the deck. Return key to close the window.
Search in Batch After open the Deck(or Graveyard), double click the card, can send this card to Temp zone without close the deck.
XYZ Summon Drag a card on other to overlay.
Show XYZ units select a XYZ monster then double it. Return key to close the window.
Token Double click empty field of monster zone. Select a token related card then double click the empty field to create the sepcified token.
Indicator Select a card in field, use volumn key to add or remove indicators.
Gravity Put up the phone, hand card face up automatically; Put down the phone, hand card face down. Can close by "feature toggle" menu.
LP Click LP then input new LP.
Change Side Enter from "Change Side" menu. Select a card to change, double click another card to switch. Return key to go back.
Deck Builder Enter from "Deck Builder" menu. Input key words to search, double click search result to add card to deck, double click the card in deck to remove. Click top and bottom part of deck, can switch between Main and Side, so that can add card into Main or Side deck. Use Volumn key to random order or sort the deck.
Change MAD Put a bg.jpg file to YGORoid/texture folder at root of SDCard.
Change Cover Put a cover.jpg file to YGORoid/texture folder at root of SDCard.
